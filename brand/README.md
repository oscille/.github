# Oscille Brand & Media Kit

Shared brand assets for Oscille projects and communications.

## Layout

- `icon/` holds the standalone mark, with `dark/` and `light/` variants in SVG and PNG.
- `logotype/` holds the mark with the Oscille wordmark, in the same variants.
- `colors.md` holds the palette and usage notes.

## Guidelines

- Prefer SVG for logos and icons; add PNG exports only when needed.
- Keep binaries small. If the kit grows to video or high-resolution assets, use Git LFS or a dedicated `oscille/brand` repo.
- Document usage rules (clear space, minimum size, dos and don'ts) here.

Add assets under `icon/`, and fill in `colors.md`.
