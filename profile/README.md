Oscille builds tools and infrastructure for engineers, researchers and teams. Our mission is to create software that solves hard problems simply, giving teams the building blocks to work faster without trading away control.

## What We Do

Oscille develops developer tools, runtimes, and platforms. We focus on the systems that sit underneath the products people build, the layers that need to be fast, reliable, and easy to integrate.

## Our Repositories

Our repositories host the source code for the various projects under Oscille. Each repository includes detailed documentation and guidelines for contribution.
