Oscille is a research company building the seed, the smallest structure a business needs to run itself with no human in operational control. The goal is generality, a seed that can run any business, not one tuned to ours. We run our own company as an instance of the seed and improve it by running it.

Read [The Oscille Constitution](https://github.com/oscille/.github/blob/main/CONSTITUTION.md) for what we are building and why.

## Repositories

Our repositories hold the projects under Oscille, each with its own documentation and contribution guidelines.
