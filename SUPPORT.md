# Support

Need help with an Oscille project?

- Check the repository's `README.md` and docs first.
- For a bug or a feature idea, open an issue using the templates.
- For a security issue, follow [`SECURITY.md`](SECURITY.md) and do not file a public issue.
- For general questions, email us at hello@oscille.com.
- For product feedback or ideas about how we can improve, email feedback@oscille.com.

Support is best effort. We respond when we can.
